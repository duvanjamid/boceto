WireScript

Un lenguaje de texto para diseñar wireframes de interfaces. Escribe, comparte y renderiza prototipos navegables sin abrir Figma.

DSLWireframes IA nativaMulti-tema Plugins
✦

IA integrada

Describe cambios en lenguaje natural y la IA reescribe el wireframe.

⬡

Texto plano

Un archivo .wire. Sin XML, sin JSON, sin herramientas propietarias.

→

Navegable

Los botones navegan entre pantallas. Prototipo click-through incluido.

◻

Multi-tema

paper · blueprint · sketch · noir. Cambia con una línea.

Instalación

App web (sin instalación)

Abre app.wirescript.dev y empieza a escribir. No requiere cuenta.

VSCode

Busca WireScript en el marketplace, o instala manualmente:

ext install wirescript.wirescript-vscode

Docsify

<!-- en index.html de tu proyecto docsify -->
<script src="wirescript-docsify.js"></script>

Prism.js (cualquier sitio estático)

<script src="prism.js"></script>
<script src="wirescript-prism.js"></script>

Inicio rápido

Crea un archivo mi-app.wire:

// Mi primera app con WireScript
theme paper

@Login
  nav MiApp
  # Bienvenido
  p Inicia sesión para continuar
  field Email
  field Contraseña *
  btn Entrar > Dashboard

@Dashboard
  nav MiApp · Inicio · Salir
  # Hola de nuevo
  ghost Cerrar sesión > Login
💡 Cada @Pantalla es navegable. Los botones con > crean flujos interactivos.

Temas

Declara el tema al principio del archivo (antes de cualquier @pantalla):

theme blueprint

@MiPantalla
  # Hola
paper — cálido, por defecto
blueprint — técnico
sketch — limpio
noir — oscuro

Plugin VSCode

El plugin añade resaltado de sintaxis, snippets y preview para archivos .wire.

// .vscode/extensions.json
{
  "recommendations": ["wirescript.wirescript-vscode"]
}

O instala la gramática manualmente copiando wirescript.tmLanguage.json a tu extensión.

⚠ El plugin de marketplace está en beta. Para instalación manual, incluye el .tmLanguage.json en el contributes.grammars de tu package.json.

Sublime Text / TextMate

Copia wirescript.tmLanguage.json a:

# macOS
~/Library/Application Support/Sublime Text/Packages/WireScript/

# Linux
~/.config/sublime-text/Packages/WireScript/

Reinicia Sublime y selecciona WireScript como lenguaje en la barra inferior.

CodeMirror 6

import { wirescript } from "wirescript-cm";
import { EditorView, basicSetup } from "codemirror";

new EditorView({
  extensions: [basicSetup, wirescript()],
  parent: document.getElementById("editor")
});

Docsify

Añade el plugin en tu index.html de Docsify:

<script src="//cdn.jsdelivr.net/npm/docsify/lib/docsify.min.js"></script>
<script src="wirescript-docsify.js"></script>

Luego en cualquier archivo Markdown de tu documentación:

## Flujo de autenticación

```wire
theme paper

@Login
  nav MiProducto
  # Iniciar sesión
  field Email
  field Contraseña *
  btn Entrar > Dashboard
```

El plugin renderizará un wireframe interactivo inline, con navegación entre pantallas.

Con IA

WireScript está diseñado para ser generado y modificado por LLMs. Incluye el DSL en el system prompt:

You are a wireframe expert. Use WireScript syntax:
  @PageName — defines a screen
  # H1 / ## H2 / p text / note hint
  ---                    — divider
  nav Logo · Link · Link — navbar
  field Label            — input (* = password, ? = optional)
  area / pick / check / toggle
  btn Label > Page       — primary button with nav
  ghost Label > Page     — outline button
  link Label > Page      — link
  kpi Value Label        — big metric
  grid Col1 · Col2       — table
  list · Item1 · Item2   — list
  row / card / aside     — layout containers (indent children 2 spaces)
  theme paper|blueprint|sketch|noir

Reply ONLY with WireScript code, no markdown, no backticks.
💡 La app oficial incluye un asistente IA en la pestaña ✦ que ya tiene este prompt configurado.
