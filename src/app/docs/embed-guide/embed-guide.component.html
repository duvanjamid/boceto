Editor

Aprende a integrar prototipos interactivos en tus blogs, documentación o herramientas internas mediante el uso de iframes.

1. El código base

Utiliza el siguiente fragmento HTML para insertar un playground. Solo necesitas reemplazar el parámetro w con tu código DSL codificado en Base64.

{{ iframeCode }}

2. Configuración (Query Params)

Personaliza cada inserción mediante los parámetros de la URL:

Parámetro Descripción Valores
mode Controla la complejidad de la interfaz. lite, preview, full
theme El color de la interfaz (shell) del editor. dark, light
readonly Si es true, el visitante no podrá editar el código. true, false
w Tu código DSL codificado en Base64. String
